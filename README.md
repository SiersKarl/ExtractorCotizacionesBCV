# ExtractorCotizacionesBCV
Programa que extrae los precios de la bolsa de Caracas y los almacena en un archivo CSV.

El objetivo es automatizar el proceso de revision de precios de las acciones para que se haga la parte tediosa en el menor tiempo posible. Con eso en cuenta, es importante destacar que dicha data puede usarse para diferentes destinos dependiendo de los analisis que deseen realizar.
